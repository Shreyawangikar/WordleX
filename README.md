# WordleX

A Wordle solver built using React, TypeScript, and Vite.

## Features
- Helps solve Wordle puzzles
- Interactive UI
- Fast performance with Vite

## Tech Stack
- React
- TypeScript
- Vite

## Setup Instructions

Clone the repository and run locally:

```bash
git clone https://github.com/Shreyawangikar/WordleX.git
cd WordleX
npm install
npm run dev
